# Jetpack Jumper

A simple jump and run game </br>
![screenshot](res/GameScreenshot1.PNG)

## Controls

Up - Fly </br>
Left/Right - Walk </br>
F - Fullscreen </br>
Escape - Exit Game

## Build

### Windows

#### Requirements

GNU GCC MinGW64

#### Compile Game

run build.bat

#### Compile Resources

run compile_res.bat
